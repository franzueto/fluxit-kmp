# FluxIt — Architecture Decisions

> **Pointer file.** All ADRs live in
> [`/plan/00_DECISIONS.md`](../plan/00_DECISIONS.md), which is the single
> source of truth. This file exists only so external readers landing in
> `/docs/` can find the log without learning the `/plan/` convention.
>
> When the ADR file passes ~15 entries (or an ADR needs diagrams /
> appendices), it migrates to per-file format under `/plan/adr/NNN-slug.md`.
> Update this pointer at that point.

- **Accepted ADRs:** see the top of [`/plan/00_DECISIONS.md`](../plan/00_DECISIONS.md).
- **Anticipated ADRs** (placeholders for upcoming phases): see the bottom of the same file.
- **Format:** `Status / Date / Context / Decision / Consequences / Alternatives considered`. Accepted ADRs are immutable — supersede with a new entry that references the old one.
